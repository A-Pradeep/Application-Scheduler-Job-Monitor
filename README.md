# Getting Started with Application Scheduler & Job Monitor.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run dev`

Runs the app in development mode.
